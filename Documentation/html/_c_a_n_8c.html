◆ CAN0_init()

void CAN0_init ( void  )

This function initializes CAN0 in CANFD mode with BRS initially off. Message buffers are set to 64 data-bytes, however when using the transmit function, the DLC is set depending on the length of the provided data. Keeping the message buffers at 64-bytes avoids accidental overwrites of future message buffers.
